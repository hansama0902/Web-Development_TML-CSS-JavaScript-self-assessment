10 Most Useful HTML Tags

  • <div>: A block-level container to group elements together.
  • <span>: An inline container for specific styling or scripting.
  • <p>: Represents a paragraph of text.
  • <a>: Used to create hyperlinks.
  • <img>: Embeds images into the document.
  • <header>: A semantic tag for the top section of a page or section.
  • <footer>: A semantic tag for the bottom section of a page or section.
  • <section>: Defines a thematic grouping of content. Commonly includes a heading, such as <h1> or <h2>, to describe the section's purpose.
  • <nav>: Represents a section of a page that links to other pages or sections within the page.
  • <article>: Represents a self-contained composition in a document, page, application, or site, which is intended to be independently distributable or reusable.

Font Size Demonstration

  • 8px: Absolute size. Commonly used for captions or footnotes, where the text does not need to scale.
  • 12px: Absolute size. The default size for most body text in web design for easy readability.
  • 1em: Relative size. Matches the font size of the parent container. Ideal for responsive designs.
  • 1.2em: Relative size. Often used for subheadings or text that needs to stand out slightly.
  • 110%: Relative size. Scales the font relative to its parent size. Good for subtle emphasis.
  • larger: Relative size. Automatically adjusts based on the surrounding text size, useful for dynamic elements like buttons.
